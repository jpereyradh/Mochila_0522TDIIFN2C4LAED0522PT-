a. ¿Quién inventó el sistema de control de versión Git y por qué?

Fue desarrollado por Linus Torvalds en el 2005, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.

b. ¿A quién pertenece actualmente Github y por qué?

Actualmente el propietario es Microsoft, con el fin de enfocarse y adquirir en su catálogo el desarrollo que generan los desarrolladores. 

c. ¿Hay otra forma que no sea la terminal para trabajar con Github?

Sí, a través de la aplicación de escritorio de GitHub
