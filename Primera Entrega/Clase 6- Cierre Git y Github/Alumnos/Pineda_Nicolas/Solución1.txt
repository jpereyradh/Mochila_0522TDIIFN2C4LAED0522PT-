a. ¿Quién inventó el sistema de control de versión Git y por qué?
Git es un proyecto de código abierto maduro y con un mantenimiento 
activo que desarrolló originalmente Linus Torvalds, el famoso creador del 
kernel del sistema operativo Linux, en 2005.
Git tiene la funcionalidad, el rendimiento, la seguridad y la flexibilidad que 
la mayoría de los equipos y desarrolladores individuales necesitan.

b. ¿A quién pertenece actualmente Github y por qué?
