Cheatsheet consigna2_clase6, Git y Github

git init: incializar un repositorio local.
gir add filename: añadir archivos al staging area.
git status: conocer el estado de los archivos.
git log: historial de los commit realizados. 
git commit -m "mensaje": realizar confirmación de los cambios en el repositorio.

git remote add origin main url_repo: conectar el repositorio local con el remoto.
git push origin main: subier archivos del repositorio local al remoto.
git pull origin main: bajar archivos del repositorio remoto al local.
git remote -v: estado de conexión entre el repositorio local y el remoto.
git clone url_repo: clonar un repositorio remoto en local.
