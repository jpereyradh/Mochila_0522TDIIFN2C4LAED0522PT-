a. ¿Quién inventó el sistema de control de versión Git y por qué?
R/ El sistema Git fue creado por Linus Torvalds, el mismo creador del sistema operativo Linux, con el fin de llevar un registro 
de los cambios que se efectuan cuando se trabaja de manera colaborativa en el desarrollo de un programa, software o archivo de computador.
